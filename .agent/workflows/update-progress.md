# Title: Обновление Отчета о Прогрессе

## Description
Обновление статуса шага и фиксация изменений в файле отчета (*_PROGRESS.md) в папке documentation/Plans/ согласно стандартам, определенным в 02-docs-rules.md.

## Steps
1. ASK: Запроси у пользователя:
   а) Уникальное имя задачи (как в имени файла плана).
   б) Номер шага из плана, который был только что завершен или обновлен.
   в) Статус этого шага (DONE / IN PROGRESS / SKIPPED).
   г) Краткое описание выполненных изменений и список затронутых файлов.
2. ACTION: Найди файл отчета YYYY-MM-DD_{имя задачи}_PROGRESS.md. Если файла нет, создай его (с общим статусом IN PROGRESS), добавив ссылку на план.
3. ACTION: Обнови соответствующий раздел в файле отчета, используя Структуру файла отчёта из 02-docs-rules.md. Установи новый 'Статус', добавь 'Изменения' и 'Затронутые файлы'. Обнови поле 'Последнее обновление'.
4. RESPONSE: Подтверди обновление отчета. Резюмируй общий статус задачи.
