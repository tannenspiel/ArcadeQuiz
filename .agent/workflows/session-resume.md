# Title: Резюме Рабочей Сессии

## Description
Восстановление контекста текущей сложной задачи для продолжения работы после перерыва/перезапуска, согласно правилу из 00-main-rules.md.

## Steps
1. ASK: Запроси у пользователя: "Какая сложная задача (имя файла плана) была активна перед перерывом?"
2. ACTION: Найди и прочти соответствующие файлы *_PLAN.md и *_PROGRESS.md в папке documentation/Plans/.
3. ACTION: Используя данные из отчета (*_PROGRESS.md) и плана (*_PLAN.md), определи текущий невыполненный шаг.
4. RESPONSE: Резюмируй текущее состояние задачи для пользователя:
   а) Общая цель задачи.
   б) Сколько шагов выполнено (X из Y).
   в) Какой шаг является текущим (TODO / IN PROGRESS) и его краткое описание.
5. ASK: Предложи продолжить работу: "Готовы ли вы продолжить работу над текущим шагом, или переходим к новой задаче?"
