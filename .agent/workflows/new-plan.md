# Title: Создание Нового Плана Задачи

## Description
Создание и форматирование нового файла плана (YYYY-MM-DD_task-name_PLAN.md) в папке documentation/Plans/ согласно стандартам, определенным в 02-docs-rules.md.

## Steps
1. ASK: Запроси у пользователя:
   а) Полное описание задачи/фичи.
   б) Краткое, уникальное имя задачи для файла (например, 'refactor-health-system').
   в) Приоритет (HIGH/MEDIUM/LOW).
2. ACTION: Используя текущую дату, сформируй имя файла: YYYY-MM-DD_{имя задачи}_PLAN.md.
3. ACTION: На основе стандартов структуры плана из 02-docs-rules.md, создай файл в documentation/Plans/. Обязательно включи разделы 'Контекст' (с указанием затронутых файлов/модулей) и 'План выполнения' (с нумерованным списком атомарных шагов).
4. ASK: Представь созданный план пользователю для согласования. Спроси: "Согласны ли вы с этим планом, или требуются корректировки?"
