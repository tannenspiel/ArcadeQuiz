# Шаблоны документов

**Версия:** 1.0  
**Дата создания:** 2025-01-26

---

## Назначение

Эта папка содержит шаблоны для создания документов проекта ArcadeQuiz.

---

## Доступные шаблоны

### TEMPLATE_MAIN.md
**Назначение:** Шаблон для создания документов основной документации.

**Использование:**
- При создании новых документов в `documentation/main/`
- Содержит полную структуру с оглавлением, примерами кода, связанными документами

### TEMPLATE_EXPORT_LLM.md
**Назначение:** Шаблон для создания упрощенных документов для LLM.

**Использование:**
- При создании новых документов в `documentation/export_llm/`
- Содержит упрощенную структуру с таблицами и структурированными данными

### TEMPLATE_TEMP_DOCS.md
**Назначение:** Шаблон для создания временных документов и отчетов.

**Использование:**
- При создании отчетов в `documentation/temp_docs/`
- Содержит структуру отчета со статистикой и результатами

---

## Правила использования

1. **Всегда используйте шаблоны** при создании новых документов
2. **Следуйте структуре шаблона** для единообразия документации
3. **Адаптируйте шаблон** под конкретные нужды документа
4. **Обновляйте шаблоны** при изменении стандартов документации
