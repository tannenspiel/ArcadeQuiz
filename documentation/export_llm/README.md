# export_llm/ - Документация для LLM

**Версия:** 2.0
**Дата:** 2026-01-28

---

## Назначение

Эта папка содержит упрощённые версии документации, специально подготовленные для быстрого ознакомления ИИ-ассистентов со структурой проекта.

---

## Принципы

1. **Минимум текста** - Только ключевые факты
2. **Структурированные данные** - Таблицы и списки
3. **Максимальная ясность** - Без лишней информации

---

## Файлы

| Файл | Описание |
|------|----------|
| `PROJECT_MAP.md` | Карта проекта (быстрый поиск компонентов и файлов) |
| `QUICK_START.md` | Быстрый старт (запуск, технологии, структура) |
| `README.md` | Этот файл (описание папки) |
